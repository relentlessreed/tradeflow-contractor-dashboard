- `homepage.png`
- `dashboard.png`
- `leads.png`
- `jobs.png`
- `estimate-builder.png`
- `invoices.png`

- `/`
- `/dashboard`
- `/dashboard/leads`
- `/dashboard/jobs`
- `/dashboard/estimates/new`

Suggested workflow:

1. Run `npm run dev`
2. Open the app in a desktop browser
3. Capture clean screenshots at consistent width
4. Save them here with the exact names above
